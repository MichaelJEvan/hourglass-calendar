Description:
Hourglass Calendar is a personal countdown calendar that keeps your most important events front and center — with live ticking countdowns, not just dates.
Track birthdays, vacations, deadlines, concerts, and anything else that matters. Set alerts and get notified with a repeating sound and flashing clock until you dismiss it. Organize events across multiple color-coded calendars.
The built-in Astro Calendar auto-populates with sunrise and sunset times, moon phases, solstices, meteor showers, eclipses, and supermoons — calculated with precision astronomical algorithms based on your location.

## Features:

- Live countdown timers — days, hours, minutes, seconds
- Multiple calendars with color coding
- Recurring events and date ranges
- Alert sounds with 15-second repeat until dismissed
- Analog clock with multiple styles and Zulu time mode
- Astro Calendar — sunrise, sunset, moon phases, and more
- Automatic data saving with export and import backup
- Universal — runs natively on Apple Silicon and Intel Macs
